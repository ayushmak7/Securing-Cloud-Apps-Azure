<h1>Securing Cloud Applications with Microsoft Azure</h1>


<h2>Description</h2>
I built, hosted and designed a cyber-blog web application using Azure’s Cloud services and Docker. Created and stored SSL certificates in Azure’s Key Vault, and bound them to secure the web application and lastly, protected the web application by utilizing Azure’s Security features, such as Azure’s Front Door, WAF, and Security Center.
<br />


<h2>Languages and Utilities Used</h2>

- <b>HTML</b> 
- <b>PHP</b>

<h2>Environments Used </h2>

- <b>Microsoft Azure</b>
- <b>Docker</b>
